
### Artify 

An online art-sharing platform where artists can upload and
display their creative works, explore other artists’ galleries, curate
favorites, and connect through appreciation and interaction. The
website emphasizes creativity, modern UI, and clean navigation for art
lovers.
### 📝 Project Overview

This is the frontend of Artify, an interactive art sharing platform. Users can explore artworks, manage their personal gallery, like and favorite artworks, and manage their profile. The frontend is built with React and styled using Tailwind CSS and DaisyUI.

### Key Sections:

- Browse news by category

- CRUD operations

- Favorite/unFavorite and like/unlike system for artworks

- Dark/light theme toggle with persistent preference

- User authentication using Firebase


## Tech Stack

**Client:** React, TailwindCSS,React Router , DaisyUI, Marquee,React – UI library, Firebase, React Toastify, React Tooltip, React Fast Marquee, React Simple Typewriter, SweetAlert2




## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`VITE_FIREBASE_API_KEY`
`VITE_FIREBASE_AUTH_DOMAIN`
`VITE_FIREBASE_PROJECT_ID`
`VITE_FIREBASE_STORAGE_BUCKET`
`VITE_FIREBASE_MESSAGING_SENDER_ID`
`VITE_FIREBASE_APP_ID`


##  🛠️ How to Run the Project Locally

Follow these steps to run the project on your local machine:

### 📌 1. Clone the repository
```bash
git clone https://github.com/rakib120822/artify-client.git
```

### 📌 2. Navigate to the project folder
```bash
cd artify-client
```
### 📌3. Install dependencies
```bash
npm install
```
### 📌 4. Run the local development server
```bash
npm run dev
```
### 📌5. Open in your browser
```bash
http://localhost:5173

```
